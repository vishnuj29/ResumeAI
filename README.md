# ResumeAI 🚀

An AI-powered Resume Builder that helps users create professional, ATS-friendly resumes in minutes. ResumeAI leverages artificial intelligence to generate, optimize, and customize resumes based on user skills, experience, and career goals.

## ✨ Features

- 🤖 AI-generated resume content
- 📄 ATS-friendly resume templates
- 🎨 Modern and responsive UI
- ⚡ Real-time resume preview
- 📥 Export resume as PDF
- 🔍 Resume optimization suggestions
- 🌙 Dark/Light mode support
- 📱 Mobile-friendly design

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI Integration
- OpenAI API / Gemini API

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/resumeai.git
cd resumeai
```

### Install Dependencies

```bash
# Frontend
cd client
npm install

# Backend
cd ../server
npm install
```

### Environment Variables

Create a `.env` file inside the server folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
OPENAI_API_KEY=your_api_key
```

### Run the Application

```bash
# Start Backend
cd server
npm run dev

# Start Frontend
cd client
npm run dev
```
